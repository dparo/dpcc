# dpcc: DParo's own C-alike Compiler frontend

![Example of running a DPL script](./gifs/showoff.gif)



## Building

- `Linux` or `MacOS`
- `GCC` or `Clang`
- `CMake >= 3.11`
- `Flex`
- `Bison`

Please refer to [this guide to build the project](https://github.com/dparo/dpcc/wiki/clone-and-build)
